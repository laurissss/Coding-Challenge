# Coding-Challenge

Installationsanleitung

  

1.

`git clone https://github.com/laurissss/Coding-Challenge.git`

  

2.

`cd coding-challenge`

  

3.

`composer install`

  

4.

`npm run install`

  

5.

`npm run build`

  

6.

`php artisan storage:link`

  

7.

Datenbankserver starten

  

8.

`php artisan migrate:fresh --seed`

  

9.

`php artisan serve`
